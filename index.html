Tugas

DDL(Data Definition Language)

1.Create table = Digunakan untuk membuat tabel baru.
Contoh syntax = CREATE TABLE [schema, ] table ( column datatype [DEFAULT expr][,...]);

2.Drop table = Digunakan untuk menghapus tabel.
Contoh syntax = Drop tabel mahasiswa;

3.Alter = Digunakan untuk mengubah struktur tabel.
Contoh syntax = Alter TABLE Buku ADD Penulis Varchar(100); Alter TABLE Buku Drop Column Judul_buku;

DML(Data Manipulation Language)

1.Insert table = Digunakan untuk menambahkan/memasukan data ke dalam tabel.
Contoh syntax = INSERT INTO nama_tabel(field ke-1, ….. field ke-n) VALUES(nilai_field_ke-1, …. nilai_field_ke-n)

2.Update = Mengubah data yang akan dirubah.
Contoh syntax = UPDATE nama_tabel SET nama_field = data_baru WHERE nama_field = data_lama;

3.Delete = Digunakan untuk menghapus data
Contoh Syntax = DELETE FROM nama_tabel [ WHERE kondisi];

4.Select tabel = Digunakan untuk menampilkan tabel.
Contoh syntax = Select * from [nama tabel];
